  Built using Core Java, OOP concepts, and CLI interface.  
</p>

---

## 🧠 Features

- 📘 Add new books
- 📖 Display all books
- 📕 Issue a book
- 📗 Return a book
- 🔎 Search books by title
- 🧮 Count total books
- 🔒 Simple CLI-based UI
